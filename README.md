## My-Tech-Stack-Learning-Hub 💃🏻🗂️ 

A personal knowledge base where I collect notes, concepts, examples, and quick references across the tech stack — all in one place.

---

## What This Repo Is

This is my running collection of learning material. It's not always a polished practice project — most of the time it's a **quick-reference notebook** I can come back to when I need to remember how something works, check syntax, or revisit a concept without Googling from scratch.

Content here includes:

- 📝 **Personal notes** — things I've learned and written down in my own words
- 📖 **Notes from sources** — summaries and key takeaways from courses, books, docs, and tutorials
- 💡 **Concepts** — explanations of how things work under the hood
- ✅ **Tasks and exercises** — practice problems and mini-challenges
- 🔬 **Examples** — small, focused code snippets demonstrating a specific idea
- ⚡ **Quick references** — syntax cheatsheets and command references I reach for often

## How I Use This

This repo is **for me first**. The goal is that if I haven't touched a topic in three months, I can open the relevant folder and be back up to speed in minutes — without digging through bookmarks or re-reading full documentation.

Some files here are thorough write-ups. Others are just a few bullet points or a code block I didn't want to lose. That's intentional — the bar for committing something is "will I want to find this later?" not "is this publication-ready?"
